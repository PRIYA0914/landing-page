# Shrekium Technologies - Landing Page (Frontend)

This is the React frontend for the Shrekium Technologies startup landing page, built with the MERN stack.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm start
   ```

The app will be available at [http://localhost:3000](http://localhost:3000).

---

## Project Structure
- App.js: Main landing page component
- index.js: Entry point
- index.css: Basic styles
